# Support

Use the right path for the right problem.

| Need | Route |
|---|---|
| Security issue | `architects@zer0pa.ai` |
| Packaging or repo-boundary problem | open a private repo issue |
| Evidence/path dispute | open a private repo issue with exact file references |
| Legal or licensing question | `architects@zer0pa.ai`; `LICENSE` is the legal source of truth |

Fastest triage happens when the report includes:
- exact file path
- exact command
- exact failure output
- exact artifact or proof reference
